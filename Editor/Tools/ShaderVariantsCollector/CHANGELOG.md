## [1.0.0] - 2024-01-01

### 新增
- 初始版本发布
- 实现基本的变体收集和管理功能
- 支持手动添加Shader和变体
- 提供引用查找功能
- 支持TreeView显示变体层级结构
- 实现进度条显示收集进度
- 支持新变体的确认/忽略操作
- 提供批量操作功能（确认全部/忽略全部）
- 支持Shader文件索引定位
- 实现变体引用查找（Material和Prefab）
- 提供关键字选择对话框
- 支持反射获取ShaderVariantCollection内部数据

### 技术特性
- 使用Unity Editor API实现编辑器扩展
- 采用TreeView实现层级显示
- 使用反射技术访问内部数据
- 支持AssetDatabase进行资源查找
- 提供完整的错误处理机制
